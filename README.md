# 🐻‍❄️ 북극곰 — 세대 계승 생존 게임

산업혁명이 시작된 **1750년부터 2050년까지**, 점점 녹아가는 빙하 위에서 북극곰 한 마리를 새끼에서 성체로 키우고, 그 새끼로 대를 이어 혈통을 존속시키는 생존 게임.

**성장할수록 강해지지만, 무거워진 몸이 디딜 얼음은 줄어드는** 모순이 핵심 긴장이다. 기후변화를 주제로 한 짧은 프로토타입.

## 🧊 게임 규칙

- **빙하 위**: 안전. 스태미나 회복.
- **물속(헤엄)**: 스태미나 소모. 0이 되면 익사.
- **에너지(체지방)**: 시간이 지나면 감소(물속·추위에서 더 빨리). 사냥에 성공하면 회복. 0이 되면 아사.
- **고립**: 올라탄 빙하가 다 녹았는데 닿을 얼음이 없으면 죽는다.

### 성장 4단계

새끼 → 청소년 → 성체 → 노년. 자랄수록 크고 사냥은 강해지지만,
**성체가 되면 몸이 무거워 작은 빙하를 밟으면 가라앉는다.**

### 세대 계승

- 성체가 되어 **짝(다른 북극곰)** 을 만나면 새끼가 태어난다.
- 부모가 죽으면 조작이 새끼로 넘어가고, 새끼는 다시 처음부터 자란다. (세대 +1)
- 새끼를 남기지 못하고 죽으면 **혈통 단절 — 게임 오버.**
- 짝은 연도가 올라갈수록 점점 드물게 나타난다.

## 🌍 시대에 따라 변하는 세계

- **빙하 융해가 가속한다.** 1750~1850 안정 → 1950~2000 가속 → 2000~2050 급격한 붕괴. 빙하는 점점 작게, 빨리 녹는다.
- **인간 위협**
  - 19세기 이전: **원주민**이 작살을 들고 접근(곰보다 느려 도망 가능). 성체 곰은 쫓아낼 수 있다.
  - 1800년부터: **총잡이** 등장. 연도가 올라갈수록 더 자주, 더 강한 총을 든다.
  - 인간은 육지를 선호하고, 용감한 일부만 물을 아주 느리게 건넌다.
- **오염**: 후반부엔 바다에 **비닐봉투**가 떠다닌다. 먹으면 건강만 나빠진다.
- 연도가 흐를수록 바다 색이 미묘하게 탁해지고, 대륙의 눈이 녹아 드러난다.

## 🏁 두 가지 결말

1. **생존** — 2050년에 살아있는 곰이 도달하면 발동. 텅 빈 바다, 마지막 작은 빙하 하나, 홀로 선 곰.
   > *"당신의 혈통은 살아남았습니다. 1750년부터 2050년까지, 300년을. 하지만 아무리 둘러봐도, 다른 북극곰은 한 마리도 보이지 않습니다."*

   이긴 것 같지만 진, 씁쓸한 엔딩.
2. **혈통 단절** — 새끼를 남기지 못하고 죽으면 발동. 죽은 연도를 보여주고 다시 시작.

## 🛠 기술

- **단일 HTML 파일**, Vanilla JS + HTML5 Canvas. 외부 라이브러리·이미지·빌드 도구 0.
- 모든 그래픽은 캔버스 도형으로 그림. 앰비언트 사운드는 Web Audio로 생성.
- 밸런스 수치(이동속도, 융해 계수, 게이지 감소율, 사냥 성공률, 짝 등장률 등)는 코드 상단 `C` 상수 객체에 모아 튜닝 가능.
